O que é?

O HTML (HyperText Markup Language) é o componente básico de um site, onde se forma toda a sua estrutura, é onde se organiza todas as categorias do site deis de cabeçalhos, rodapés, títulos, parágrafos e entre outros conceitos. A sigla HTML em português fica “Linguagem de Marcação Hipertexto”, ou seja, o HTML não é uma linguagem de programação e sim de marcação onde permite inserir conteúdos e estabelecer a estrutura básica do site.

Como funciona?

O HTML funciona da seguinte maneira, ele precisa da tag HTML que fica acima de todos, a cabeça (head) e o corpo (body) essas são as principais tags que são utilizadas para a estrutura de um site em HTML.


Como são feito os códigos HTML?

Os códigos em HTML eles são compostos por TAGS que são fechadas por < >, a partir dessas tags conseguimos usar a linguagem para a criação de títulos, cabeçalhos, rodapés e etc.



Principais tags do HTML

As principais tags do html estão listadas abaixo:

  • <DOCTYPE html>/</html> - Significa que a sua página será no formato HTML5
  • <head>/</head> - a área onde ficarão as tags mais importantes.
  • <body>/</body> - a estrutura do seu site.
  • <h1>//<h1> - Cria um título.
  • <h2>/</h2> - um subtítulo.
  • <p>/</p> - Começara um novo parágrafo.
  • <b>/</b> - Deixa em negrito o que estiver dentro dela.

Vantagens do HTML

A vantagem de se usar o HTML é que a partir dele você consegue dar a estrutura esquelética do seu site, colocando títulos, parágrafos, imagens, vídeos e entre outros.

Conclusão

E essa foi a conclusão que chegamos sobre o HTML, espero muito que você tenha gostado de conhecer um pouco sobre o HTML e espero você logo abaixo para conhecer o CSS!




O que é?

Como foi dito anteriormente o HTML é a estrutura de um site, já o CSS (Cascading Style Sheets) ele vem para complementar o HTML, ou seja, ele vai estilizar os elementos escritos em HTML, ou seja, ele é a parte da decoração do seu site, dar cor e vida ao seu site.

Como HTML tem a sua própria sigla o CSS também tem a sua, em português ela fica “Folha de Estilo em Cascatas”, como dito anteriormente ele é o que vai dar cor ao seu site.

Como funciona

O CSS (Cascading Style Sheets) ele funciona de maneira que você consiga alterar o layout de uma página rapidamente, definindo cores e fontes por exemplo. Não só isso, ele também ajuda a diminuir a repetição de conteúdo na estrutura do código, e ele trabalha em conjunto com o HTML, permitindo várias criatividades de design em layout.

Códigos CSS

Os códigos em CSS são usados para dar estilo a sua página de site, eles não são complicados, precisa de um pouco de criatividade e então você consegue deixar sua página web muito mais bonita visualmente.

Principais configurações do CSS

As configurações do são divididas em 3 categorias: CSS Inline, CSS Interno Externo.

CSS Interno

No CSS Interno é utilizado as tags <style> na seção <head> de seu arquivo HTML, esse estilo de CSS é muito efetivo para estilizar logo de vez uma única página a única desvantagem é que sua página após carrega-lá ela irá demorar um pouco mais.
